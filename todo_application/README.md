# Todo Application

This is a simple **Todo Application** built using **Flask**, where you can manage tasks with basic features like adding, updating, and deleting tasks.

## Features

- Add new tasks.
- Mark tasks as completed.
- View tasks with their details.
- Delete tasks.

## Installation

### Prerequisites

Ensure you have the following installed:

- Python (preferably version 3.7 or higher)
- pip (Python package installer)
